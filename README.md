# SocialBox-Termux
SocialBox is a Bruteforce Attack Framework [ Facebook , Gmail , Instagram ,Twitter ] , Coded By BlackMamba
# Installation
```
apt-get update
apt-get install git
git clone https://github.com/OfficialBlackMamba/SocialBox-Termux.git 
cd SocialBox-Termux
chmod +x install-sb.sh
./install-sb.sh
```
# Run
```
./SocialBox.sh
```
